_Example of code:_

React-component Button, that change the text on button: 

```javascript
import React from 'react'

class Button extends Component {
    state = {
        isOpen: false
    }
    checkState = () => {
        if (!isOpen) {
            this.setState({
                isOpen: !isOpen
            })
        } 
    }
    render() {
        return(
            <React.Fragment>
                <button onClick={this.checkState()} >
                    {this.state.isOpen ? 'Hide' : 'Open'}
                </button>
            </React.Fragment>
        )
    }
}
```
